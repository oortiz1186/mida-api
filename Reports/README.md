# Reportes

Esta carpeta contendrá la lógica para reportes consumidos por Soporte MIDA y aplicaciones Android.

Ejemplos de reportes:

- Clientes con saldo.
- Documentos por cliente.
- Ventas por periodo.
- Cobranza pendiente.
- Productos más vendidos.

Los reportes deben exponerse por endpoints controlados y reutilizar servicios de integración o sincronización.
